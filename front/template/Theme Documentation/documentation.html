HTML Structure

The template is based on Bootstrap 3 Framework - http://getbootstrap.com 

<div class="row">
  <div class="col-md-12">
        Level 1 column
        <div class="row">
            <div class="col-md-6">Level 2</div>
            <div class="col-md-6">Level 2</div>
        </div>
    </div>
</div>
Please note: You can use all Classes and Scripts from Bootstrap 3, they will all work! Not only the grid!

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. If you need more information about the Main Grid, please visit this site: http://getbootstrap.com/css/#grid

CSS Structure

The organization of the CSS is one of our priorities. These are the CSS file we're using in the template.

Customization and Styles

General

This Theme has various Style options for different Kinds of Layouts.

Colors

All the Colors can be changed in the CSS File color-default.css. Or include one of our predefined Stylesheets into the Head Section of the Theme.

Fonts

To use another Font, please choose your new Font at the Google Font Site, include the given Style Tag inside the Head Section of the Theme and add the CSS Markup to smarton.css.

In the default Theme we have used this Code for example:

<link href='http://fonts.googleapis.com/css?family=Raleway:400,100,200,300,500,600,700,800,900' rel='stylesheet' type='text/css'>

Layout

The Main Layout of the Theme can easily be changed by adding or removing one CSS Class to the Body.

Use this for the Wide Version:

<body>
  <div class="page-wrapper">
    ...
  </div>
</body>

Use this for the Boxed Version:

<body>
  <div class="page-wrapper boxed">
    ...
  </div>
</body>

Backgrounds (for boxed Version only)

You can define the Background for the Boxed Version by adding a class to the Body.

Available Classes for the Body are bg-pattern1 - bg-pattern20.

<body class="bg-pattern1">
  ...
</body>

Headers

SmartOn is providing 6 types of top pages headers that can be combine how you think will be better for you.

Check the demo to see the headers in action. Simply add one of the following classes to your .header-wrapper: .light-top-header, .center-top-header. Using this 2 classes and the default boostrap classes we can have the headers in diffrent styles (see demo).

<div class="header-wrapper light-top-header center-top-header">
...
</div>

Footers

SmartOn is providing 5 types of footer pages that you can use as you wish.

The same rule as the header apply here but the difference is that here we don't have special classes for the footer. Just by using the Boostrap you can build your own footer. We made 5 types of footers for you so you can have a base to start.

Menus

SmartOn is providing 6 types of menu. They can be combine and customized as you wish with over 20 different results.

The same as the headers we have special classes for the menu that need to be added to .header-bottom. Use the following classes:.menu-full-width, .menu-tab, .menu-radius, .menu-separator. You can combine this classes to build other styles of the menu. Also you can add icons the the menu using fontawesome.

Inner Headers

SmartOn is providing 5 types of inner pages headers that can be combine in over 25 variations.

The inner header where we have the title page and the breadcrumbs can be styled using Boostrap classes and some special classes. You can add those classes to .page-header. Check the demo to see what can you build.

Sliders

We use two Main Sliders in this Theme, Revolution Slider jQuery Plugin and the Flexslider 2 jQuery Plugin.

Revolution Slider

Revolution Slider is our Main Slider with a lot of options to customize, it includes full Layer Support, Videos, Transitions and much much more. The Revolution Slider must be initialised in the jQuery File application.js. To see the full Documentation of this awesome jQuery Slider, please look here:

Documentation

Flexslider 2

Flexslider is the slider you'll find in the content. Flexslider is a simple slideshow script but with lots of options. The Flexslider must be initialised in the jQuery File application.js. You can find more Documentation here:

Documentation

Lightbox

The Lightbox which is integrated in the Theme runs with the Magnific Lightbox, a awesome jQuery Plugin to handle different Media Types. The Lightbox is initialised in the application.js. For more Informations please have a look at the Documentation:

Documentation

Maps

All Maps in this Theme are driven by the free Service from Google, GoogleMaps.

To customize the maps simply open application.js in the js folder and starting with the line 265 you can fiind the maps script. Change the coordinates of the map at line 271 and 281 with your own coordinates. Also change the coordinates at line 326 so the script can center the marker. To change the content of the pop up go to line 307. To edit the marker just replace the image in img/theme/google-marker.png.

Contact Form

We have added a working PHP Contact Form in our Theme.

Important You need to make sure your server supports PHP 5.0 + otherwise the dynamic contact form will not work. Please note that if you deploy it locally, you will also need a local server to check the dynamic contact form.
Open the file contact/process-form.php. Following is the screenshot of the values you need to adjust 

		require_once "class.phpmailer.php";
		$mail = new PHPMailer();
		$mail->IsMail();
		$mail->IsHTML(true);    
		$mail->CharSet  = "utf-8";
		$mail->From     = $email;
		$mail->FromName = $contactName;
		$mail->WordWrap = 50;    
		$mail->Subject  =  $contactName;
		$mail->Body     =  "Full Name: " . $contactName. ".
";
		$mail->Body     .= "Message: " . $message. ".";
		$mail->AddAddress('cosminiorgus@gmail.com');
		$mail->AddReplyTo($email);

Edit the e-mail address on line 62 and set your own one. This will be the address where the e-mails sent from contact form will come. You can also set multiple e-mails by separating them with commas.
The subject for e-mail sent can also be set by modifying line 60.
If you need more modifications, like changing the success/error messages, you should edit file contact/process-form.php as well as contact.php and change the values in corresponding places. Just browse through the abovementioned files to locate those fields.

Twitter

We have added a working Twitter Feed in our Theme, you'll need to register a App @ Twitter.

We are using jQuery Tweetie - Simple Twitter Feed Plugin to have twitter display on our theme.New Twitter API requires oAuth Token Key, so it’s three step process. Here are the steeps to make it work:

1. First, you need a consumer key and secret keys. Get one from dev.twitter.com/apps.

2. Edit js/themejs/api/config.php file and replace variables with your Consumer and oAuth Keys.

<?php
    // Consumer Key
    define('CONSUMER_KEY', 'CONSUMER_KEY_HERE');
    define('CONSUMER_SECRET', 'CONSUMER_SECRET_HERE');

    // User Access Token
    define('ACCESS_TOKEN', 'ACCESS_TOKEN_HERE');
    define('ACCESS_SECRET', 'ACCESS_SECRET_HERE');
?>
